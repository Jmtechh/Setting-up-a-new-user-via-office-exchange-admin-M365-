<h1>Setting up a new user via office exchange admin M365</h1>


<h2>Description</h2>
The aim of this lab is to demonstrate how to create a new user using Microsoft 365 exchange admin.

<h2>Utilities Used</h2>

- <b>Microsoft 365 admin</b>


<h2>Environments Used </h2>

- <b>Windows 10</b>

<h2>Program walk-through:</h2>

The first step I took was to click on ‘Add a User Template.’ This enabled me to assign predefined attributes to an individual user by applying a template with preset qualities.
